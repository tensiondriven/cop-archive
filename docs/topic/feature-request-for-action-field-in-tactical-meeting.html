Feature Request for 'ACTION' field in Tactical Meeting

I have often found that when facilitating new groups with a novice Secretary, it can take the Secretary a while (like a number of meetings) to remember how to assign a meeting output as a Project/Trigger/Tension rather than action Action, from the dropdown box that says 'Action'.

It would be REALLY helpful if this box was a different colour or had some quick and easy way to distinguish it from anything else in that area so that instead of saying 'you see that box that says action about three sevenths of the way from the left-hand side of your screen and about four nineteenths of the way from the top next to where it says...'  and instead I could say 'that pink box that says Action is a drop down list, you can choose Project from there'.
