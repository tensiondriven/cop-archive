Holacracy Implementation Roles App v.01

Create the following roles in the client organization's GCC or other appropriate circle:

Name: Holacracy Lead

Accountabilities:

  • Triaging all Holacracy implementation related questions, and communicating with internal and external coaches as needed to select appropriate support services
  • Leading and supporting the implementation of Holacracy
  • Liaising with HolacracyOne to plan and adjust service usage as appropriate

Name: Implementation Logistics

Accountabilities:

  • Supporting any logistical requirements of the Holacracy implementation, and helping to resolve any obstacles
  • Planning and executing resources to align office and home work environment with virtual meeting requirements

Name: Holacracy Coach

Accountabilities:

  • Supporting colleagues by answering Holacracy practice questions
  • Facilitating circle meetings and providing Holacracy coaching upon request
