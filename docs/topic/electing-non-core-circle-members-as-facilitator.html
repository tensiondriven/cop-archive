Paragraph 2.5.1 in the Constitution states only Core Circle Members can be elected into each of the Elected Roles. However, we see merit in electing skilled facilitators from outside the circle to help the circle improve its Holacracy practice. In fact one of our circles has done so, not realizing this is violates the Constitution, and it seems to be bringing the expected benefit. Any views on this? Anybody else explored this option?
