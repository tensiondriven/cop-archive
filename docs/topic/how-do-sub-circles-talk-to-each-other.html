We understand the function and accountabilities of the Rep Link for these new sub-circles to the main circle. What we can't figure out is how these 3 sub-circles communicate with each other. Do we create a policy to invite and/or request them to be present at tactical and governance meetings of the other subcircles? What are our next steps? Thanks! Morgan
