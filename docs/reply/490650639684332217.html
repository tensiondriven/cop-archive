Also if a proposal is not escalated, it would be usefull if it was automatically accepted in glassfrog.

Are there any functions in glassfrog that would help?

Yes there is a function in GlassFrog. It's a parameter for each circle you have to set up accordingly with a policy which determines the number of days for answer. If no answer after the specified delay and no request for escalating, GlassFrog will do the update automatically.

Click on the icon next to the name of the circle, a window will be opened by GlassFrog and you may add the number of days required.
